# Sadiasdev

Portfolio web construit avec Vite, React et TypeScript.

## Prerequis

- Node.js 18+
- npm

## Installation

```sh
npm install
```

## Lancer en local

```sh
npm run dev
```

## Build production

```sh
npm run build
npm run preview
```

## Stack technique

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
